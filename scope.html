Web Serial needs a Chromium-based browser (Chrome / Edge / Brave / Chromium) and a secure context. The simplest reliable way: drop this file in a folder, run python3 -m http.server there, and open http://localhost:8000/scope.html. Click Connect and pick the ESP32 port. Trigger locks the waveform on a rising zero-crossing so steady tones sit still instead of scrolling.
